ENTREGA DE MATERIAL

This page will handle material delivery. You'll need to implement logic for selecting materials from stock, assigning them to a recipient, and updating stock levels. This will likely involve a new ManagedBean (e.g., `EntregaMaterialBean`) and potentially a `Movimentacao` entity if not covered by `SolTransporte` or `SolManu`.

Consider:

  • Listing available materials.
  • Input fields for recipient, quantity to deliver, date.
  • Updating the material's `quantMat` in the database.
  • Perhaps creating a `Movimentacao` record for auditing.
